# Parsowanie HTML-a LLM-em

## Instalacja:
- Zainstalować środowisko z `requirements.txt`
- Zainstalować [Ollama](https://ollama.com/) 
- Uruchomić: 
  - `ollama create Gemma_html_parser -f Gemma_html_parser.model`
  - `ollama create Gemma_html_parser_large_ctx -f Gemma_html_parser_large_ctx.model`
